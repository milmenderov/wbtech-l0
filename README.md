## App - L0
Сервис для записи в базу заказов и быстрого их просмотра
### Необходимые переменные для запуска:
| Name        | Default value |
|-------------|---------------|
| env_name    | env_value     |
### Как запускать
Для старта прописать команду(она сбилдит и запустит необходимые контейнеры)   
```make re```
